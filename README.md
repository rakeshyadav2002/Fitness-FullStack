\# Fitness Full Stack Application



A full-stack Fitness Management web application built using Angular, ASP.NET Core Web API, SQL Server, Entity Framework Core, and JWT Authentication.



\## Live Project Links



Frontend Website:

http://fitnessfullstackui.runasp.net/



Backend Swagger API:

http://fitnessfullstack.runasp.net/swagger/index.html

\## Project Modules



\### Authentication Module



Users can register and login securely. After login, a JWT token is generated and used for protected API calls.



\### Dashboard Module



The dashboard displays the main user interface after login.



\### BMI Module



Users can calculate and manage BMI records.



\### Calorie Tracker Module



Users can manage calorie tracking details.



\### Diet Plans Module



Users can view diet plans from the database.



\### Profile Module



Users can view their profile details.

\## How to Run Project Locally



\### Backend



1\. Open the backend project in Visual Studio.

2\. Update the connection string in `appsettings.json`.

3\. Run the SQL Server database script.

4\. Run the ASP.NET Core Web API project.

5\. Open Swagger and test the APIs.



\### Frontend



1\. Open the Angular project in Visual Studio Code.

2\. Install packages:



```bash

npm install

```



3\. Run Angular project:



```bash

ng serve

```



4\. Open browser:



```text

http://localhost:4200

```
